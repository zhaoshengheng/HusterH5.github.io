# 校园地图图片

本目录存放路线页使用的两张校园手绘地图。

## 当前文件

| 文件名 | 用途 | 图片尺寸 | 展示方式 |
|--------|------|---------|---------|
| `Main-map.jpg` | 主校区地图 | 1672 × 941（比例 1.777） | `aspect-ratio: 1672/941` 自适应高度，`100% 100%` 填满 |
| `TJ-map.png` | 同济校区地图 | 1024 × 682（比例 1.501） | `background-size: cover` 填满容器 |

## ⚠️ 文件名大小写敏感

主校区地图文件名为 **`Main-map.jpg`**（大写 M），CSS 引用路径必须完全匹配：

```css
.campus-map[data-map="main"]::before {
  background-image: url("assets/maps/Main-map.jpg");
}
```

> **历史教训**：曾因 git 跟踪为小写 `main-map.jpg` 导致本地（Windows 不区分大小写）正常、但 GitHub Pages（Linux 区分大小写）404 地图不显示。提交时务必确认大小写一致。

## 地图展示逻辑（v60）

### 主校区
- 容器使用 `aspect-ratio: 1672 / 941` 让高度随宽度自动按图片比例变化
- `background-size: 100% 100%` 完整填满容器
- 效果：横向铺满，纵向无空白，地标按钮坐标不变

### 同济校区
- 容器固定高度，`background-size: cover` 填满并裁切边缘
- 效果：地图填满容器，上下无空白

## 地标按钮定位

地标按钮（`.map-label`）使用百分比坐标定位，存储在 `script.js` 的 `landmarks` 数组中：

```js
{ id: 1, name: '南大门—毛主席像', x: 50, y: 82, ... }
```

- `x`：水平百分比（0~100，相对地图容器宽度）
- `y`：垂直百分比（0~100，相对地图容器高度）
- `size`：可选，按钮直径（默认 64px，大号 82px）

更换地图图片后，如果新地图的建筑位置有变化，需要同步调整 `x` / `y` 坐标。
