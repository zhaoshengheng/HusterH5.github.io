# 时空喻园 

本版针对用户反馈做保守修复：

1. 修复身份页流程：
   - 进入身份页后，必须先选择一种“入园方式”
   - 选择后再点击“确认入园并进入路线图”才能进入后续页面
   - 已毕业校友 / 社会大众卡片间距重新拉开
   - 选择身份后，卡片不再重新布局或再次乱动

2. 动效调整：
   - 组件入场速度放慢
   - 部分组件从左侧进入，部分从右侧进入
   - 切换身份时不触发整页二次乱动

3. 路线页优化：
   - 删除原先地图下方那组数字地标组件
   - 改为直接通过地图上的标记点选地标
   - 保留路线预览卡，用于显示当前选中地标

4. 封面可读性增强：
   - 标题区增加更清晰的暗色玻璃底板
   - 提升文字对比度，减轻颜色遮挡问题

5. 点击/遮挡修复：
   - 强化装饰层 pointer-events 处理
   - 提升交互按钮与卡片的层级

6. 身份风格区分：
   - 高中生 / 在读生 / 校友 / 社会大众四种身份，会在后续路线、地标页、集章页拥有不同的轻度观赏风格背景

## v10 热修复

修复点击“随机漫游 / 上一个 / 下一个”后，地标页变成空白浅黄色背景的问题。
原因是刷新地标类型 class 时误清除了当前详情页的 active/page-ready 状态，本版已保留页面状态。
